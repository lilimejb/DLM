Это приложение для заказа еды через Колл центр. Позвонивший говорит своё имя и фамилию, после чего пользователь вносит эти данные в БД и оформляет заказ.
Приложение может:
	Вносить данные о позвонившем в БД
	Оформлять заказ
	Показывать заказы

Краткое руководство по использованию можно получить, открыв окно "Справка"

Ссылка на GitHub - https://github.com/lilimejb/DLM

					!Важно!
			Программа запускается через main.py
			Нельзя оформлять заказ без регестрации
